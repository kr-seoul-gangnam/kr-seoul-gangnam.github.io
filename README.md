# 강남구 · 서울특별시

> 강남구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-gangnam.github.io/**

강남구는 대한민국을 대표하는 상업·업무·교육의 중심지로, 테헤란로를 축으로 한 업무지구와 압구정·청담·신사동의 명품 상권이 세계적 인지도를 지닌 곳입니다. 1970년대 이후 이른바 '강남 개발'을 통해 논밭이던 한강 이남 땅이 계획적으로 조성되면서 대한민국 도시화의 상징이 되었습니다. 코엑스와 삼성동 국제업무지구, 대치동 학원가, 도산대로 갤러리 거리가 한 구 안에 공존하며 낮과 밤의 표정이 뚜렷이 다릅니다. 부동산 가격과 교육열이 전국에서 가장 높은 지역으로 자주 언급되어 '강남'이라는 이름 자체가 하나의 브랜드처럼 통용됩니다. 봉은사와 선릉·정릉 같은 조선의 유산이 초고층 빌딩 사이에 자리해 역사와 첨단이 겹쳐 보이는 독특한 경관을 만듭니다.
